			======================================================
					Inner OR Nested LOOPs
			======================================================
=>The process of Defining One loop inside of Another Loop is Called Inner OR Nested Loop
=>The Execution Process of Inner loops is that "For Each value of Outer Loop, inner Loop Repeated for Finite Number of Times until Inner loop becomes False".
=>In Python Programming, we can use Inner loops in 4 ways. They are
-------------------------------------------------------------------------------------------------------------------------------------------------------------
Approach1:  for loop in for loop
-------------------------------------------------------------------------------------------------------------------------------------------------------------
		for varname1  in Iterable-Object-1:   # outer Loop
			----------------------------------
			for varname2 in Iterable-Object-2:  # Inner Loop
			      ----------------------------------------
			      ----------------------------------------
			else:
			      ----------------------------------------
		else:
			---------------------------------------------
			----------------------------------------------
-------------------------------------------------------------------------------------------------------------------------------------------------------------
Approach2:  while loop in while loop
-------------------------------------------------------------------------------------------------------------------------------------------------------------
			--------------------------------------
			while(Test Cond1):  # outer loop
			       ----------------------------
			       ----------------------------
			       while(Test Cond2):  # inner loop
			              -------------------------
				      -------------------------
				else:
				      ------------------------
			else:
				-------------------------------

-------------------------------------------------------------------------------------------------------------------------------------------------------------
Approach3:  for  loop in while loop
-------------------------------------------------------------------------------------------------------------------------------------------------------------
			--------------------------------------
			while(Test Cond1):  # outer loop
			       ----------------------------
			       ----------------------------
			      for varname2 in Iterable-Object-2:  # Inner Loop
					----------------------------------------
					----------------------------------------
			     else:
					----------------------------------------
			else:
				-------------------------------

-------------------------------------------------------------------------------------------------------------------------------------------------------------
Approach4:   while loop in for  loop 
-------------------------------------------------------------------------------------------------------------------------------------------------------------
		for varname1  in Iterable-Object-1:   # outer Loop
			----------------------------------
			----------------------------------
			 while(Test Cond2):  # inner loop
			              -------------------------
				      -------------------------
				else:
				      ------------------------
		else:
			---------------------------------------------
			----------------------------------------------
------------------------------------------------------------------------------------------------------------------------------------------------------------
