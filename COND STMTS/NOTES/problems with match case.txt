				================================================
						match case statement
				================================================
=>match case statement is one of the New Features added from Python 3.10 Version Onwards.
=>The purpose of match case statement is that "To Implement Menu Driven Applications OR Pre-Designed Conditions".

-------------------------
Examples
-------------------------

				==================================================
					Arithmetic Operations
				==================================================
					1. Addition
					2. Substraction
					3. Multiplication
					4. Division
					5. Mod Div
					6. Exponentiation
					7. Exit
				=================================================
				Enter Ur Choice:
				=================================================
		ch=int(input("Enter UR Choice:"))
		match (ch):
		     
		     case 1:
				# Add Code
		     case 2:
				# Substraction Code
		     case 3:
				# Mul Code
		     case 4:
				# Div Code
		     case 5:
				# mod div Code
		    case 6:
				# Exp Code
		    case 7:
		             # exit code
		   case _ :   # deafult case block
		         print("Ur Selection of Operation is wrong--try again")
